# MusicPlayer - 网易云音乐第三方 Android 客户端

基于 Android 原生开发的网易云音乐第三方客户端，使用 UnblockNeteaseMusic 代理服务转发 API 请求。

## 技术栈

- **语言:** Java
- **网络:** Retrofit 2.9 + OkHttp 3.14 + Gson
- **播放:** MediaPlayer（音频）+ Media3 ExoPlayer（视频）
- **图片:** Glide 4.15
- **存储:** SQLite + SharedPreferences
- **UI:** Material Design、ViewPager2、RecyclerView

## 运行要求

- Android Studio 最新版
- JDK 17+
- Android SDK 34

## 快速开始

1. 用 Android Studio 打开本项目
2. 等待 Gradle Sync 完成（已配置阿里云镜像）
3. 点击 Run ▶
